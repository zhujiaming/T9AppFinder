# T9找应用 Android App

## 介绍

超简洁的安卓上的T9应用搜索效率工具。

- 启动轻且快
- 快速搜索定位目标应用
- 根据点击频次优先展示目标应用
- 支持展示最近使用应用
- 支持应用长按更多设置
