### T9找应用

超简洁的安卓上的T9应用搜索效率工具。

- 启动轻且快
- 快速定位目标应用
- 根据点击频次优先展示目标应用
- 支持应用常驻展示
- 支持应用长按更多设置
- 实验性便捷功能

- 非正式版本，可能存在bug，设计也不美观
